厂牌标语 / SLOGAN One Ride.
All Drift. 新启万象，万物流转。 One motion, every signal drifting. 하나의 질주, 모든 신호의 드리프트.

ORAD is NEORIGIN's label for sound, visual, and interaction experiments.

ORAD는 NEORIGIN의 사운드, 비주얼, 인터랙션 실험 레이블입니다.

MODULE 01

把 demo、噪声、节奏和人声先做成一套可变形的声音系统。

Turning demos, noise, rhythm, and voice into a mutable sound system.

데모, 노이즈, 리듬, 보컬을 변형 가능한 사운드 시스템으로 만듭니다.

用于企划前期的声音方向测试、艺人概念探索与发行前的听感校准。

Used for early sound direction tests, artist concept exploration, and pre-release listening calibration.

초기 사운드 방향 테스트, 아티스트 콘셉트 탐색, 발매 전 청감 조율에 사용됩니다.

MODULE 02

让 logo、影像、字体、舞台和网页互动保持同一个世界观。

Keeping logo, film, type, stage, and web interaction inside one world.

로고, 영상, 타이포, 무대, 웹 인터랙션을 하나의 세계관 안에 둡니다.

负责视觉规则、物料延展、回归页面和动态预告的整体方向。

Directs visual rules, asset expansion, comeback pages, and motion teasers.

비주얼 규칙, 에셋 확장, 컴백 페이지, 모션 티저의 방향을 잡습니다.

MODULE 03

把线上互动、现场灯光、音频触发和粉丝线索连接成一次体验。

Connecting web interaction, live lighting, audio triggers, and fan clues.

웹 인터랙션, 라이브 조명, 오디오 트리거, 팬 단서를 하나로 연결합니다.

用于演出、预热、隐藏线索和限时解锁机制。

Designed for shows, preheats, hidden clues, and time-limited unlocks.

공연, 프리히트, 숨겨진 단서, 기간 한정 언락에 쓰입니다.

MODULE 04

把企划过程保存成可继续生长的档案，而不是一次性宣传物。

Saving each project as a growing archive, not disposable promotion.

프로젝트를 일회성 홍보물이 아닌 계속 자라는 아카이브로 저장합니다.

适合后续纪录片、展览、网页资料库和实体发行延展。

Built for documentaries, exhibitions, web databases, and physical releases.

다큐멘터리, 전시, 웹 데이터베이스, 피지컬 릴리스 확장에 적합합니다.
